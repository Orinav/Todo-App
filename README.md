## Full Stack Mini Project(Learning purposes)
Todo App built with Flask, HTML, CSS, Vanilla JavaScript and PostgreSQL.

## How to run the Flask part
Go to backend/main.py and run it.

## How to run the Database
Currently the code makes the Database run in Docker environment, so make sure you installed Docker and have knowledge to connect the Database to it.
Alternatively you can change the code and make the Database run in different environment.

## About
This project will let you make new Todos, delete them, or mark them as completed.
